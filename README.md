## 功能模块

### 用户功能

- **语音垃圾分类**：用户可上传音频文件，系统将利用语音识别技术自动进行垃圾分类。
- **个人中心**：用户可以查看和修改个人信息，并查看历史分类记录。
- **注册与登录**：用户需要注册并登录后才能使用系统功能。

### 管理员功能

- **分类规则维护**：管理员可以新增、修改或删除垃圾分类规则。
- **用户管理**：管理员有权查看、编辑和删除用户信息。
- **系统信息管理**：管理员可更新首页信息和公告等内容。

### 扩展功能

- **分类结果追溯**：系统记录每次分类操作的详情，便于查询和追踪。
- **多语言支持**：语音识别功能支持多种语言，提高系统的适用性。

## 系统角色

- **普通用户**：可使用语音垃圾分类、查看个人信息和历史分类记录等功能。
- **管理员**：负责分类规则维护、用户管理和系统信息管理。
